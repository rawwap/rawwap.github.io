# 前言

贯穿我整个大一的智能车生活结束了，是的智能车已经成为 “生活” 了。在这段时间中，很感谢两个队友和管哥的陪伴，以及其他组一起做车的友友，大一做车的这一年很快乐、充实、精彩，很感谢渡龙的赞助，不得不说渡龙真的好有钱。最终的结果也算是不幸中的万幸，成功表演出了 3.64m/s 的小车、一个国二和 STC 发的惊喜。以下是我对整个智能车一年的总结。

# 初级训练

智能车的一切是从初级训练开始的，但由于时间线比较长，初级训练的记忆也比较少了。还能记得的是数码管和最后的 4s 小车，当时由于搜索能力可以说一窍不通，检索信息的能力比较弱，搜索的途径也就基本只有 B 站和 edge 了，当时还好多位数码管的代码不是特别复杂，我就模仿网上的代码写了一个点亮的库（当时没搜到那个库，后来才发现有这个东西）。然后就是 4s 小车，了解最多的就是对于车模结构的理解，摩擦、重心、重量、前瞻等等关键点也从一篇篇文章和测试中发掘，同时也简单的认识了 PID，大概在摸索了两三周之后成功将小车的速度提到了 4s。

# 寒假集训

集训一开始，开始了解各个车模，查找往年的技术报告、用 CoppeliaSim 仿真、抽气和风扇下压方案测试、对于车模上限的简单测试、探索日本 tracerobot 他们比赛的车模。体验 897 的上班生活，8 个人住在一起，一起看世界杯，体验一群小阳人（还记得当时有一段时间大家从早上咳嗽到晚上。挺抽象的哈哈），
